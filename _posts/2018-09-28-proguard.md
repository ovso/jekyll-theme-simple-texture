# What is proguard?

[Proguard](https://www.guardsquare.com/en/products/proguard){:target="_blank"}는 Java 바이트 코드에 가장 널리 사용되는 [옵티마이저](https://en.wiktionary.org/wiki/optimizer){:target="_blank"}다. Java 및 Android 응용 프로그램을 최대 90% 더 작고 최대 20% 빠르게 만듭니다. 또한 Proguard는 클래스, 필드 및 메소드의 이름을 모호하게 하여 리버스 엔지니어링에 대한 최소한의 보호 기능을 제공한다.
