# OctoTree

깃허브를 볼 때면 답답할 때가 있다. 트리 탐색을 할 수 없기 때문이다. OctoTree는 트리 탐색을 가능하게 한다. 정말 편하다.

# Session Buddy

크롬 브라우저에서 고정 탭을 항상 사용하고 있는 나로서는 쌍수를 들고 반길 만한 플러그인이다. 탭 설정을 저장해주기 때문이다. 이제 맘 편하게  command + q 해도 된다.
